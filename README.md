# fastenv

这是一个简单的 fastenv 程序

## install

要安装这个程序，请执行以下命令：

    brew install 

## use

安装后，你可以在终端中运行 `fastenv` 命令来查看输出：

    ./fastenv

## uninstall

要卸载这个程序，请执行以下命令：

    brew uninstall fastenv
